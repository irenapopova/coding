What are Modules in Javascript

Modules are nothing but just small javascript files. splitting your main code into small javascript files and those files are called modules

we have 2 directives keyword (import and export)

there are two types of export

>
  1. default export
  2. named export

Advantages

  • Readablity and debugging
  • isolation
  • top level scope
  • useablity
  • modules use 'use strict' mode by default
