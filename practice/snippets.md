# Cheatsheet  --- JavaScript Functions
#### Reference Cheatsheet

### ES6  Arrow function (Async function)
```javascript

const getData = async() => { 
    var data = "Hello World"; 
    return data; 
} 
  
getData().then(data => console.log(data)); 
// Output: Hello World 



```
### Await function 
```javascript

const getData = async() => { 
    var y = await "Hello World"; 
    console.log(y); 
} 
  
console.log(1); 
getData(); 
console.log(2); 

```
